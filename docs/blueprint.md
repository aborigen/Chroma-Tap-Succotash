# **App Name**: Chroma Tap

## Core Features:

- Dynamic Game Canvas: Displays the circular game track with rotating color segments and the moving white circle, forming the central game area.
- White Circle Movement & Speed Control: Animates the continuous movement of the white circle along the circular track and dynamically increases its rotation speed after a successful tap.
- Tap Detection & Validation: Detects user taps on the screen and validates if the white circle is currently positioned over a segment matching the background color.
- Score Tracking & Display: Accurately tracks the player's score, incrementing it with each correct tap, and prominently displays the current score on the screen.
- Game Timer & End Condition: Manages a countdown timer for each game session and triggers the 'Game Over' state if the timer expires or an incorrect tap is made.
- Start/Restart Game UI: Provides intuitive controls for the player to initiate a new game and restart after a game over screen, maintaining a continuous play loop.

## Style Guidelines:

- The primary color, a vibrant light blue (#26ACD9), evokes focus and dynamism, serving as a key interactive element on the bright background.
- The background color, a subtle light gray-blue (#F9FAFB), offers a clean, non-distracting canvas that highlights game elements with clarity.
- An accent color, a lively aqua-green (#17CFB5), provides contrast and draws attention to critical feedback or actionable elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics suitable for a clear, precise game interface.
- Minimalistic and clear icons for UI elements such as play/pause and game over states, ensuring quick comprehension.
- A focused and centered game area with key information like score and timer positioned for easy glanceability, maintaining a clean aesthetic.
- Smooth and fluid animations for the white circle's movement and subtle visual feedback on taps, enhancing the reactive gameplay experience.
